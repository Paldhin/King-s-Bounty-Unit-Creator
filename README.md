# King-s-Bounty-Unit-Creator

## Description

A program with the purpose of allowing the creation of fan-made units based in the RPG game "King's Bounty". 

DISCLAIMER: this program only manages text files and does not interact with the game in any way. Therefore, it is not a mod neither a fan-game.

## Relevant Information

For interested users (not developers), download the compilated "KingsBountyUnitCreator.jar" file. You can also download the files and compilate the program yourself.
